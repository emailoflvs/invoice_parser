# Установка исходного кода

В этом архиве содержится структура проекта и основные конфигурационные файлы.

## Полный исходный код

Все Python-файлы с полным кодом находятся в чате Claude по ссылке:
https://claude.ai/chat/b8356800-9477-49a7-a00b-784be32a6b97

## Быстрая установка:

1. Найдите в чате все файлы с пометкой `# file: путь/к/файлу.py`
2. Скопируйте код каждого файла в соответствующее место в проекте
3. Или попросите Claude: "создай все файлы проекта invoiceparser"

## Список файлов для копирования:

Все файлы перечислены в INSTALL_CODE.md в корне проекта.

## Альтернатива:

Вы можете попросить Claude в новом чате:
"Создай файл X.py для проекта InvoiceParser" - и он создаст нужный файл на основе предыдущих сообщений.
