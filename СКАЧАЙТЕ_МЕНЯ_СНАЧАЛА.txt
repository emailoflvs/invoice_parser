📋 ПОСЛЕ УСТАНОВКИ:

1. Настройте .env:
   cp .env.example .env
   # Добавьте GEMINI_API_KEY

2. Запустите Docker:
   docker-compose up --build

3. Используйте CLI:
   docker-compose run --rm app python -m invoiceparser.app.main_cli \
     parse --path /app/invoices/your_document.pdf
