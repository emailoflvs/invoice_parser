## 🎨 Основные элементы

### 1. Header (Шапка)
- Градиентный фон (фиолетовый → сиреневый)
- Иконка документа с AI значком
- Название и описание проекта

### 2. Upload Card (Карточка загрузки)
- Белая карточка с округлыми углами
- Drag & Drop зона с пунктирной рамкой
- Hover эффекты и анимации
- Информация о выбранном файле
- Кнопка "Распознать документ"

### 3. Progress Card (Карточка прогресса)
- Анимированный спиннер
- Progress bar с градиентом
- Процент выполнения
- Текст статуса

### 4. Results Section (Секция результатов)
- Grid layout для карточек
- Карточки с градиентными заголовками
- Таблица товаров
- JSON viewer с подсветкой синтаксиса
- Кнопки действий

### 5. Settings Modal (Модальное окно настроек)
- Центрированное модальное окно
- Поле ввода токена
- Кнопки сохранения/отмены
- Backdrop с затемнением

### 6. Settings Button (Кнопка настроек)
- Плавающая кнопка справа снизу
- Иконка шестерёнки
- Rotation эффект при hover
- Всегда доступна

## 🎭 Анимации

### Появление элементов
```
fadeInDown ↓  - Header появляется сверху
fadeInUp ↑    - Секции появляются снизу
fadeIn →      - Плавное появление
slideInUp ↑   - Модальное окно
```

### Интерактивность
```
hover        - Увеличение и подсветка кнопок
active       - Нажатие кнопок
spin         - Вращение спиннера
progress     - Заполнение прогресс бара
```

## 💡 Tips & Tricks

### Для пользователей
1. Используйте Drag & Drop для быстрой загрузки
2. Сохраните токен один раз - он запомнится
3. Скачивайте JSON для дальнейшей обработки
4. Проверяйте RAW JSON для debugging

### Для разработчиков
1. Токен хранится в localStorage
2. API endpoint: `/parse`
3. Проверяйте Network tab в DevTools
4. Логи доступны через Console
